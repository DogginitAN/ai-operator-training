# AI Operator Training Platform

Student-facing dashboard for AI Native Operator coaching. Curriculum track, kanban homework board, resource library, and core principles.

## Stack
- Next.js 14 (App Router) + TypeScript + Tailwind CSS
- localStorage for task state persistence
- Deployed on Vercel

## Student Data
All student content lives in `src/data/student.json`. To create a new student instance, fork the repo and update the JSON.

## Deploy
```bash
npm install
npm run build
# Push to GitHub, connect to Vercel
```

## Design
Inspired by Smart Learn AI Education App (Behance). Light surfaces, blue→purple gradient accents, Plus Jakarta Sans typography, soft rounded cards.
