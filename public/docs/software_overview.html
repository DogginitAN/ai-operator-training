THE BRAIN Claude Desktop

The interface most people start with — but far more powerful than a chat window. Claude Desktop has four distinct modes, each suited to different kinds of work.

CHAT MODE

What: Conversational AI — thinking, writing, analyzing, brainstorming

Use for: Strategy, document review, email drafting, research, problem-solving

Cost: Included with subscription

COWORK MODE

What: Claude controls your browser and desktop apps — clicks, navigates, reads screens

Use for: CRM lookups, form filling, GUI-based workflows, browser automation

Cost: Included with subscription

CODE TAB

What: Claude Code embedded in the Claude Desktop window — no terminal needed

Use for: Quick builds, file creation, code iteration when you don't want to open a terminal

Cost: Included with subscription

PROJECTS

What: Persistent context containers — instructions + documents that carry across every conversation inside the project

Use for: HouseAccount context, CLAUDE.md files, business-specific AI memory

Cost: Included with subscription

THE HANDS Claude Code CLI
CLAUDE CODE (TERMINAL)

What: Agentic coding loop that runs in PowerShell/Terminal. Reads your files, writes code, runs commands, builds complete software, fixes its own mistakes.

INSTALL

Windows: irm https://claude.ai/install.ps1 | iex

Mac: npm install -g @anthropic-ai/claude-code

Requires: Git, Node.js 18+

Key commands:

  • claude — start a session
  • /init — create CLAUDE.md in current project
  • /compact — compress context when window fills up
  • Shift+Tab x2 — enter plan mode (review before executing)

Cost: Included with subscription

THE MEMORY Connectors & Context

MCP connectors wire Claude into your actual tools. Install each once in Claude Desktop settings — they're available in every conversation from that point on.

  • Desktop Commander — Read/write files, run terminal commands, system info. Claude's hands on your machine.
  • Filesystem — Direct file access within allowed directories. Reads docs, CSVs, code files.
  • Google Calendar — Read schedule, create events, check availability. "What's on my calendar tomorrow?"
  • Google Drive — Access docs, sheets, slides. "Find the Q1 pricing spreadsheet."
  • Gmail — Read, search, draft emails. "Summarize my unread emails from providers."
  • Chrome Extension — Claude can see and interact with your browser tabs in Cowork mode.
Cost for all connectors: Free (included with Claude Desktop)
THE INFRASTRUCTURE Always-On (Session 5)
OPENCLAW

What: Open-source messaging gateway that connects Claude to Telegram, Discord, SMS, and more

Runs on: Mac Mini (always-on)

Cost: Free (open source)

Why: Turns Claude from "something you open on your laptop" into "something you can text from anywhere"

TELEGRAM BOT

What: Your personal AI assistant in Telegram — message it from your phone, get Claude-powered responses

Cost: Free

Requires: OpenClaw running on Mac Mini

DISCORD BOT

What: Same as Telegram but in Discord — useful if your team is there

Cost: Free
